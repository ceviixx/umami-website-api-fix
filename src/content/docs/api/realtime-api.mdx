title: Realtime

Realtime data for your website.

**Endpoints**

```text
GET /api/realtime/:websiteId
```

---

## GET /api/realtime/:websiteId

Creates a user.

**Parameters**

- `startAt`: (number) Timestamp (in ms) of start date.
- `endAt`: (number) Timestamp (in ms) of end date.
- `timezone`: (string) Timezone.
